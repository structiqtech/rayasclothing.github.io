# Rayas Clothing Website

React + Vite + Tailwind CSS site configured for GitHub Pages.

## Install

```bash
npm install
```

## Run Locally

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview Build

```bash
npm run preview
```

## Deploy To GitHub Pages

```bash
npm run deploy
```

The deploy script builds the site with the GitHub Pages base path and publishes `dist/` to the `gh-pages` branch.
